- load error message string table

=head1 SYNOPSIS

=for openssl generic

int ERR_load_LIBRARY_strings(void)

=head1 DESCRIPTION

In the description here, B<I<LIBRARY>> is used as a placeholder for any
of the sub-library parts of the OpenSSL libraries.

These functions are generated during the build process.
The generated function will be in F<I<LIBRARY>err.c>, and the error codes
will be put in F<include/openssl/I<LIBRARY>err.h>.

These functions are normally called automatically during library
initialization, and would not normally need to be used by application
code.

=head1 RETURN VALUES

The functions always return 1.
