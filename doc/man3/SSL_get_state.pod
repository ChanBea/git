=head1 NAME

SSL_get_state, SSL_in_before, SSL_in_init, SSL_is_init_finished - Query the SSL handshake state

=head1 SYNOPSIS

 #include <openssl/ssl.h>

 OSSL_HANDSHAKE_STATE SSL_get_state(const SSL *ssl);

 int SSL_in_before(SSL *s);

 int SSL_in_init(SSL *s);

 int SSL_is_init_finished(SSL *s);

=head1 DESCRIPTION

SSL_get_state() gets the current B<ssl> handshake state.

SSL_in_before() checks if an B<ssl> handshake has started.

SSL_in_init() checks if an B<ssl> handshake is in progress.

SSL_is_init_finished() checks if an B<ssl> handshake has completed.

=head1 NOTES

These methods provide details about the SSL state machine's current state.

=head1 RETURN VALUES

SSL_get_state() returns the current B<OSSL_HANDSHAKE_STATE> flags.
SSL_in_before(), SSL_in_init() and SSL_is_init_finished() return 1 if the condition is met, otherwise 0.
