=head1 NAME

BN_strong_lucas_prime - Checks for a strong lucas prime

=head1 SYNOPSIS

 #include <openssl/bn.h>

 int BN_strong_lucas_prime(const BIGNUM *p, BN_CTX *ctx);

=head1 DESCRIPTION

BN_strong_lucas_prime() checks taht B<p> is a strong Lucas prime using the
Selfridge parameters.

The parameter B<ctx> should be previously allocated using L<BN_CTX_new(3)>
or B<NULL>.

=head1 RETURN VALUES

Returns 1 when probably prime, 0 when it's composite, and -1 on error.

=head1 HISTORY

BN_strong_lucas_prime() was added in 1.1.x.
